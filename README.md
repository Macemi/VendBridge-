## Data Overview & Insights

### User Acquisition Insights

- **Marketing campaign and awareness are effective.**
- **However, the conversion funnel is leaking between:**
  - `Install → Sign-Up` (drop-in rate)
  - `Sign-Up → Active Retention` (increased uninstalls & crashes)
- **Improving user onboarding flow and app stability is critical to sustain growth.**

#### Key Metrics (July)
| Metric | Value | Change |
|-------|--------|--------|
| Total App Installs | 1,817,986 | ↑8.9% |
| Total User Sign-Ups | 1,152,117 | ↑7.0% |
| Sign-Up Rate | 63.37% | ↓1.7% |
| App Crashes | 36,554 | ↑8.3% |
| Avg Time Spent | **97 mins** | ↑5.4% |
| Total App Uninstalls | 13,856 | ↑13.6% |

#### User Engagement Quality
- **Average Time Spent: 97 minutes (↑5.4%)**
- Users who stay are **highly engaged**, which is a **strong signal of app value**.
- However, this is **undermined by high churn and uninstalls**.

#### Temporal Sign-Up Trends
- **Peak sign-ups around July 8**, followed by fluctuations.
- Likely tied to a **campaign or feature launch** — valuable for timing future efforts.

### Engagement Overview Insights

#### By Region
- **US leads in daily activity.**
- **LATAM is lowest** → **potential growth region.**

#### By Device
- **Android dominates usage** → indicates **platform-specific optimization opportunities.**

#### DAU Trend
- Daily activity **fluctuates but maintains a stable mid-to-high usage base.**

#### Key Metrics
| Metric | Value | Change |
|-------|--------|--------|
| Daily Active Users | 879,573 | ↑4.2% |
| Total # Transactions | 443,115 | ↑7.0% |
| Churn Rate | 1.2% | ↑6.1% |

#### User Behavior Gap
- Users are active: **browsing, searching, comparing products**
- But **fewer users completed transactions**
- This **gap between engagement and monetization** is critical — a known early-stage marketplace behavior.
- May point to:
  - Friction in transaction flow
  - Unclear value proposition
  - Trust issues
- **The stability of the average suggests consistent platform usage.**

## Recommendations

### 1. Improve Data Collection for Campaign & User Behavior Insights

> **Current Limitation:** Lack of visibility into user acquisition sources and onboarding pathways.

**Unable to identify:**
- Which marketing campaigns drove installs/sign-ups in each region
- Which sign-up methods users preferred (Email, Google, Apple ID, QR code referrals, etc.)
- Which channels contributed to **high-quality users** vs. **churn-prone users**

**This limits:**
- Accurate assessment of **campaign ROI**
- Understanding **regional adoption drivers**
- Optimizing **user onboarding flow**
- Tailoring strategies to **reduce churn & improve conversions**

---

### Action Plan

| Recommendation | Action Plan |
|---------------|-------------|
| **Optimize Conversion Funnel** | • Implement **Single Sign-On** and **Social Sign-In** to reduce onboarding friction.<br>• Launch a **guided first-time user tour** to improve activation.<br>• Use **funnel analytics** to pinpoint and address drop-off stages. |
| **Boost Transaction Efficiency** | • **A/B test transaction flows** to identify and remove friction.<br>• Introduce **micro-incentives** (discounts, loyalty points) to increase conversion.<br>• Monitor **user-to-transaction ratios** to refine monetization strategies. |
| **Enhance Regional Performance** | • Continue investment in **high-performing regions** like the **US**.<br>• **Localize campaigns, language, payment options** for **LATAM**.<br>• Deploy a **dynamic leaderboard** to gamify regional engagement. |
| **Strengthen User Retention** | • Leverage high time spent with **gamification** (badges, levels).<br>• **Segment high-engagement cohorts** and replicate their experience.<br>• Launch **re-engagement campaigns** targeting churn-prone users. |
| **Improve Technical Stability** | • Prioritize **crash diagnostics** and **device-specific optimizations**.<br>• Set **SLAs** for issue resolution (e.g., **48-hour turnaround**). |
